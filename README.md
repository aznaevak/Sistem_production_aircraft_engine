Учебный проект, посвящённый анализу и проектированию информационной системы.
## Основной фокус проекта:
- логика работы системы
- требования к системе
- моделирование данных
- сценарии взаимодействия пользователей с системой
## Задачи в рамках проекта
- Анализ предметной области и бизнес-процессов
- Определение границ системы и заинтересованных сторон
- Формирование функциональных и нефункциональных требований
- Моделирование поведения и структуры системы
- Проектирование структуры базы данных и связей между сущностями
## Артефакты проекта
- UML-диаграммы (Use Case, Sequence, Activity, Class)
- Модель данных (ER-диаграмма)
- SQL-запросы для работы с данными
- Описание требований к системе
## Используемые инструменты и технологии
- UML
- ER-моделирование
- SQL (MySQL)
